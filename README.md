# aiden.dev

Personal portfolio website for Aiden — 17 y/o dev.

## Features

- Live Discord presence via [Lanyard](https://github.com/phineas/lanyard) WebSocket
- Dynamic theming — background, cards, and banner color extracted from Discord avatar automatically
- MP4 video banner with smooth loop fade and pfp color overlay
- Activity cards showing current Spotify / game status
- Projects showcase with modal view and roadmap tabs
- Friends page (WIP)
- Multiple themes (midnight, orange, rose, blue, etc.)
- Star canvas animation
- Easter egg

## Stack

- Vanilla HTML / CSS / JS
- [Lucide Icons](https://lucide.dev)
- [Lanyard API](https://api.lanyard.rest)
- Google Fonts (DM Mono, Familjen Grotesk)
